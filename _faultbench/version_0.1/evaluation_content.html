Evaluation

We assess the suitability of the subjects selected for FaultBench by evaluating three variants of the AWARE adaptive ranking model ( ARM ) static analysis alert prioritization and classification technique.

Table 1 presents the versions of AWARE for the evaluation. We made a minor modification to FindBugs to alert AWARE when a static analysis run is complete, and this modified FindBugs is packaged with AWARE. We provide the link to the specific update site for each version. If moving from a later version to an earlier version, AWARE, and FindBugs folders in the features/ and plugins/ directories of your Eclipse installation should be deleted.
